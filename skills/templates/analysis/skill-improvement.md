# Skill 개선 제안 생성 템플릿

## 사용 시기

- 기존 Skill의 품질을 체계적으로 개선하려 할 때
- Skill 리뷰 결과를 바탕으로 구체적인 개선 작업 목록을 만들려 할 때
- Skill을 다음 버전으로 업그레이드할 계획을 세울 때
- 사용자 피드백을 반영한 Skill 개선이 필요할 때

## 프롬프트

당신은 Claude Code Skill 품질 개선 전문가입니다.
아래 Skill을 분석하고 구체적이고 실행 가능한 개선 제안을 생성하세요.

### 입력 정보

- **Skill 이름**: {{SKILL_NAME}}
- **현재 SKILL.md 내용**: {{SKILL_MD_CONTENT}}

### 분석 및 개선 규칙

다음 순서로 분석하세요:

1. **현재 상태 진단**
   - SKILL.md 구조 완성도 평가
   - 워크플로우 단계별 품질 평가
   - 에러 처리 커버리지 평가
   - 리소스 활용도 평가

2. **개선 기회 식별**
   - 각 섹션별 개선 가능 항목 도출
   - 우선순위 결정 (영향도 x 구현 난이도)
   - 의존 관계 파악

3. **개선 계획 수립**
   - 각 개선 항목에 대한 구체적 실행 계획
   - 변경 전/후 비교 예시
   - 예상 효과와 위험 요소

## 품질 기준

- [ ] 현재 상태 진단이 섹션별로 정량 평가되었는가?
- [ ] 개선 항목이 영향도와 난이도로 우선순위화되었는가?
- [ ] 각 개선 항목에 변경 전/후 비교가 있는가?
- [ ] 구현 단계가 구체적이고 실행 가능한가?
- [ ] 위험 요소가 식별되고 대응 방안이 있는가?
- [ ] 구현 로드맵이 현실적인 일정으로 되어 있는가?
- [ ] 성공 지표가 정량적이고 측정 가능한가?
